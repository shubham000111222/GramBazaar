# Functional Requirements

## User Management
- Users must be able to register and login
- Password reset and OTP-based verification

## Seller Module
- Sellers can register and upload documents
- Sellers can add, edit, and delete products
- Sellers can track orders and sales

## Buyer Module
- Buyers can browse products
- Buyers can search and filter products
- Buyers can place orders and make payments

## Admin Module
- Admin can approve or reject sellers
- Admin can moderate products
- Admin can manage users and reports

## Order & Payment
- Secure payment gateway integration
- Order tracking and status updates
